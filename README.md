## Steps

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.1.0.

Clone the repository:
```bash
git clone https://github.com/jabg97/gmstek_test.git
```

Install dependencies:
```bash
npm install
```

Run DEV server:
```bash
ng serve
```
